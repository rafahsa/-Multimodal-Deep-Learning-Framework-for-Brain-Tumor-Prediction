# Experiments Directory

This directory contains experiment tracking and results.

## Structure

- `resnet50_3d/`: ResNet50-3D experiment results
- `swin_unetr/`: Swin UNETR experiment results
- `mil/`: MIL experiment results

## Recommended Contents per Experiment

- Model checkpoints
- Training curves and metrics
- Validation results
- Test set predictions
- Configuration snapshots (for reproducibility)
- Logs and notes

## Best Practices

- Use timestamps or experiment IDs for organization
- Save configuration files used for each experiment
- Document any deviations from standard procedures
- Keep raw metrics and processed visualizations
